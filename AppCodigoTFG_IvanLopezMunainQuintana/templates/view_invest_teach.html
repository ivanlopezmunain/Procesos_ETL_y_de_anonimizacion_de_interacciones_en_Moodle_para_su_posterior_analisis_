Access Moodle DB


Learning analytics

This option of the application is associated with the use by university teachers in order to monitor their subjects. It will allow downloading the data associated with their subjects and obtaining a dashboard of them. This information will not be anonymized because the data is already known by the teacher due to his situation. The required credentials are the URL where you log into Moodle and its username and password.

Research

This option of the application is associated with the use of academic data for research purposes. Due to this, all the data will be anonymised and it only be possible to view/download the data, so the researchers can work directly with them. The required credentials are the same as in the other mode (username, password and login URL), besides the API access token and the corresponding URL.
